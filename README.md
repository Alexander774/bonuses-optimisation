# Bonuses Optimisation

This project explores how loyalty bonus points can be used strategically over a sequence of purchases.

The main goal is to minimise total spending by choosing when to save bonuses and when to spend them. The optimisation problem is solved using dynamic programming.

## Files

- `Bonuses Optimisation.ipynb` — main notebook with the model, simulations, and preliminary results.

## Method

The project models each purchase as a decision point. At each step, the customer can either save bonuses or spend available bonuses. The optimiser finds the strategy that minimises total cost over the full purchase sequence.
